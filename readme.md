# Privacy Control Mechanism (PCM) — Data Repository

This repository hosts the dataset and supporting materials for the **Privacy Control Mechanism (PCM)** project, which examines privacy disclosures and user-facing privacy controls in conversational AI platforms. The data supports multiple research questions (RQ1–RQ3) in the broader study of privacy documents, mechanisms, and user practices within these platforms.

---

## Repository Structure

### 1. **Codebook (`codebook..xlxs`)**
Defines the coding schema used in the project. 

### 2. **Annotation File (`annotation.xlsx`)**
Contains the full set of human annotations for all PCMs.

### 3. **Screenshots (`/screenshots/`)**
A folder of screenshots that associated with the PCM annotation.

### 4. **Personas (`RQ3_Personas and Questions.docx`)**
Personas and questions that are used specifically to support RQ3 (user experience experiment).
